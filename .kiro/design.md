# SheAffirm - Design Document

## Overview
SheAffirm is a simple, calming web application designed to provide daily positive affirmations to boost confidence and positivity for women.

## Visual Design

### Color Palette
- Primary: Soft lavender (#E6D5F5)
- Secondary: Gentle pink (#FFE5EC)
- Accent: Warm coral (#FFB5A7)
- Text: Deep purple (#4A3B5C)
- Background: Cream white (#FFF9F5)

### Typography
- Primary font: System fonts (Arial, Helvetica, sans-serif)
- Affirmation text: Large, centered, easy to read
- Button text: Medium weight, clear

### Layout
- Single-page application
- Centered content with maximum width for readability
- Card-based design for affirmation display
- Soft shadows and rounded corners
- Responsive design for mobile and desktop

## User Interface Components

### Main Affirmation Card
- Large centered card with soft shadow
- Daily affirmation text prominently displayed
- Gentle gradient background
- Rounded corners (16px)

### Action Buttons
- "New Affirmation" button - primary action
- "Save to Favorites" button - secondary action
- Soft hover effects
- Rounded pill-shaped buttons

### Favorites Section
- Collapsible list below main card
- Each favorite displayed in a smaller card
- Remove button for each favorite
- Smooth animations

## Interaction Design
- Smooth fade transitions when changing affirmations
- Gentle hover effects on interactive elements
- Visual feedback for saved favorites
- No harsh animations or jarring transitions
